## Card props

|Prop|Type|Required|Default|
|----|----|--------|-------|
|[title](#title)|string|true|n/a|
|[subtitle](#subtitle)|string|false|n/a|
|[headingLevel](#heading-level)|number|false|3|
|[href](#href)|string|true|n/a|
|[image](#image)|string|true|n/a|
|[children](#children)|node|false|n/a|
|[renderAnchor](#renderanchor)|node|false|n/a|

### title

*Type*: string

The title should briefly describe where the Card will take the user when they click on it.

### subtitle

*Type*: string

An optional addition to the Card heading. Only use if necessary.

### headingLevel

*Type*: number, default: 3

The heading level used for the title.

### href

*Type*: string

Provide a url for where this card should route to.

### image

*Type*: string

Provide a url to an image. This is purely decorative and shouldn't be required to understand the Card.

### children

*Type*: node

Regular React children.

### renderAnchor

*Type*: node

An optional parameter to allow overriding the anchor rendering. Useful for using Card along with react-router or other client side router libraries.
